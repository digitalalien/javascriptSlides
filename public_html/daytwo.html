Operators

Basic Operators in JavaScript allow you to manipulate values.

  • = is used to assign a value
  • + is used to add values
	x = 1;
	y = 5;
	z = x + y; //result of x will be 6

Note that in JavaScript when you try to add a number and a string you may get unexpected results. 

	var myNum = 1;
	var myString = "5";
	alert(myNum + myString); //this will alert 15 not 6
	
	//You can force a string to act like a number by using the Number Constructor
	alert(myNum + Number(myString)); //this will alert 6
	
	//Or using the Unary Plus operator
	alert(myNum + +myString);

Arithmetic Operators

Arithmetic Operators are used to perform arithmetic between variable and/or values.

(y = 5)
+ Addition x = y+5 x = 10
- Subtraction x = y-5 x = 0
* Multiplication x = y*5 x = 25
/ Division x = y/5 x = 1
% Modulus (division remainder) x = y%2 x = 1
++ Increment x = ++y x = 6 and y = 6
x = y++ x = 5 and y = 6
-- Decrement x = --y x = 4 and y = 4
x = y-- x =5 and y = 4
Assignment Operators

Assignment Operators are used to assign values to variables.

(y = 5 and x = 10) x = y
= x = 5
+=x += y x = 15
-=x -= y x = 5
*=x *= y x = 50
/=x /= y x = 2
%=x %= y x = 0
