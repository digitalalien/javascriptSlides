Operators

Basic Operators in JavaScript allow you to manipulate values.

  • = is used to assign a value
  • + is used to add values
	x = 1;
	y = 5;
	z = x + y; //result of x will be 6

Note that in JavaScript when you try to add a number and a string you may get unexpected results. 

	var myNum = 1;
	var myString = "5";
	alert(myNum + myString); //this will alert 15 not 6
	
	//You can force a string to act like a number by using the Number Constructor
	alert(myNum + Number(myString)); //this will alert 6
	
	//Or using the Unary Plus operator
	alert(myNum + +myString);

Excercise 4 Answer

var placeOrderBtn = document.getElementById('placeOrder');
placeOrderBtn.addEventListener('click', getOrder, false);

function validateOrder(sandwich, quantity, price) {
    if (sandwich == '' || quantity == '' || price == '') {
        return false;
    }
    return true;
}

function getOrder() {
    var sandwich = document.getElementById('sandwich').value;
    var quantity = document.getElementById('quantity').value;
    var price = document.getElementById('price').value;
    if (validateOrder(sandwich, quantity, price)) {
        var total = price * quantity;
        var sentence = "Order: " + sandwich + " Quantity: " + quantity + " Price: $" + price + " Total: $" + total;
        var displayOrderDiv = document.getElementById('yourOrder');
        displayOrderDiv.innerHTML = sentence;
    } else {
        alert('All fields are required');
    }
}
Objects

Everything in JavaScript is an Object: a String, a Number, an Array, a Date, etc.

In JavaScript, an object is data, with properties and methods.

  • Properties - Variables created inside an object.
  • Methods - Methods are actions that objects can perform.

Properties:

car.name = Kia

car.model = Optima

car.hp = 270

car.color = black

Methods

car.start()

car.drive()

car.stop()

car.accelerate()

Creating Objects

There are three ways to create an object. The one you use will depend on the situation.

Example 1

You'll want to create your object in this way if you need to create several similar objects.

function Car(name, model, hp, color){
    this.name = name;
    this.model = model;
    this.hp = hp;
    this.color = color;
    this.start = function(){
        alert(this.name +' ' + this.model + ' has started.');
    }
}

car1 = new Car('Kia', 'Optima', '270', 'Black');
car1.start();

car2 = new Car('Ferrari', '458', '562', 'Red');
car2.start();

Accessing Properties

Using "dot notation"

var Car = {
    name: 'Kia',
    model: 'Optima',
}

var myCar = Car.name;

Using "bracket notation"

var Car = {
    name: 'Kia',
    model: 'Optima',
}

var myCar = Car['name'];

What's the difference?
"dot notation" properties must be valid JavaScript identifiers (a sequence of alphanumeric characters including the "_" and "$") and cannot start with a number. "bracket notation" property is a string and does not have to be a valid identifier, they can have any value.
