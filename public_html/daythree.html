What is jQuery?

jQuery is "The Write Less, Do More, Javascript Library".

The purpose of jQuery is to make it easier to use JavaScript on your website. jQuery takes common tasks that require many lines of JavaScript and wraps it into methods that you can call with a single line of code.

jQuery also simplifies things like: AJAX calls and DOM manipulation.

Why jQuery?

While there are plenty other JavaScript frameworks to choose from, jQuery is the most popular and also the most extendable.

Many of the biggest companies use jQuery

Excercise 3 - Events

Get the values from input boxes and call function to display order

  • Create an input box for Sandwich, Quantity and Price
  • Create a "Place Order" button
  • When the button is clicked call a new placeOrder function, that gets the input values and displays the order in a div
Excercise 3 Answer
<p>Sandwich:<input type='text' id='sandwich'/></p>
<p>Quantity:<input type='text' id='quantity'/></p>
<p>Price:<input type='text' id='price'/></p>
<button id='placeOrder'>Place Order</button>
<div id='yourOrder'>Order Not Placed</div>

<script>
var placeOrderBtn = document.getElementById('placeOrder');
placeOrderBtn.addEventListener('click', getOrder, false);

function getOrder(){
    var sandwich = document.getElementById('sandwich').value;
    var quantity = document.getElementById('quantity').value;
    var price = document.getElementById('price').value;
    
    var total = price * quantity;
    var sentence = "Order: " + sandwich + " Quantity: " + quantity + " Price: $" + price + " Total: $" + total;
    
    var displayOrderDiv = document.getElementById('yourOrder');
    displayOrderDiv.innerHTML = sentence;
}
</script>
